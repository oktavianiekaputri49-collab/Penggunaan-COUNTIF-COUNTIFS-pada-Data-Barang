📊 Penggunaan COUNTIF dan COUNTIFS pada Data Barang
Deskripsi
Project ini berisi pengolahan data barang menggunakan Microsoft Excel dengan fokus pada penggunaan fungsi COUNTIF dan COUNTIFS. 
Analisis dilakukan untuk menghitung jumlah data berdasarkan kriteria tertentu secara lebih spesifik.

Dataset
Data yang digunakan meliputi:
- Kode barang
- Jenis barang
- Ukuran
- Harga (Normal/Diskon)
- Stok

Rumus yang Digunakan
Beberapa fungsi Excel yang digunakan:
- COUNTIF → menghitung jumlah data dengan satu kriteria  
- COUNTIFS → menghitung jumlah data dengan lebih dari satu kriteria  

Analisis yang Dilakukan
- Menghitung jumlah barang berdasarkan jenis (misalnya Shampoo, Sabun, dll)  
- Menghitung jumlah barang berdasarkan status harga (Normal/Diskon)  
- Menghitung kombinasi kriteria (misalnya jenis barang tertentu dengan ukuran tertentu)  

Tujuan
Project ini dibuat untuk:
- Memahami penggunaan COUNTIF dan COUNTIFS  
- Melatih pengolahan data berbasis kriteria  
- Meningkatkan kemampuan analisis data dasar menggunakan Excel  

Pengembangan Selanjutnya
Project ini masih dapat dikembangkan dengan:
- Menambahkan visualisasi data  
- Menggunakan dataset yang lebih kompleks  
- Menggabungkan dengan fungsi Excel lainnya  

Penutup
Project ini merupakan bagian dari proses belajar saya di bidang statistika dan analisis data.
